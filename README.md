# 🐶🐱 Friends

<br />

### 각 지역의 유기동물 입양 정보와 실종 동물의 정보를 조회할 수 있는 서비스입니다.

---

'반려동물 사지말고 입양합시다' 라는 말을 많이들 들어봤으나 어디서 정보를 찾고 어떻게 입양을 해야하는지 모르시는 분들이 많습니다.
<br />

Friends에서는 전국의 유기동물과 보호소의 정보를 얻을 수 있습니다.
<br />

Friends는 전국의 유기동물에게 따듯한 손길을 전해주고자 합니다.
